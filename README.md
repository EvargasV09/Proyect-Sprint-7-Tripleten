# Proyect-Sprint-7-Tripleten

Este proyecto corresponde al Sprint 7 y consiste en el desarrollo de una aplicación web interactiva para el análisis exploratorio de datos de vehículos usados. La aplicación permite a los usuarios visualizar de manera sencilla información clave del conjunto de datos mediante gráficos interactivos. A través de una interfaz web intuitiva, el usuario puede generar un histograma para analizar la distribución del kilometraje de los vehículos y un diagrama de dispersión para observar la relación entre el kilometraje y el precio. El objetivo del proyecto es aplicar herramientas de análisis de datos, visualización y desarrollo web con Streamlit, facilitando la exploración de datos y la toma de conclusiones a partir de representaciones gráficas claras e interactivas.
